Gesture-Based Musical Notation Editor

Intuitive gesture-based input for musical symbols using Java, MVC, and Swing.

Project Overview

The **Gesture-Based Musical Notation Editor** enables users to draw musical symbols through gesture-based input. Built with **Java, MVC architecture, and Swing**, this project includes a **Shape Trainer** that recognizes and adapts to user-defined gestures. The editor dynamically creates and modifies music notation symbols, improving the efficiency of digital music composition.
